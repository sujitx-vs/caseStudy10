# Social Media Sentiment Analysis using Apache Spark

## Overview

PySpark case study demonstrating distributed analytics, ETL, Spark SQL
and Spark MLlib.

## Features

-   Spark Session
-   RDD Operations
-   DataFrame Operations
-   Spark SQL
-   ETL Pipeline
-   Sentiment Classification

## Tech Stack

-   Python
-   Apache Spark 4.1.2
-   PySpark
-   Java 21

## Workflow

1.  Load CSV
2.  Clean Data
3.  RDD Operations
4.  DataFrame Analytics
5.  Spark SQL
6.  ETL
7.  ML Model

## Dataset

Social media posts with engagement metrics and sentiment labels.
